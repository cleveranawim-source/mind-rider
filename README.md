# 마음의 밤길 라이더 (Mind Ride)

야광 버섯 숲을 1인칭으로 질주하며 **감정 조절**을 몸으로 배우는 SEL 게임 프로토타입.
대상: 중·고등학생 / 영역: 마음건강(감정 조절) + 공동체(반 협동)

> 핵심 은유: **속도 = 감정의 각성도.** 빨리 달리면 마음 온도가 오르고 시야가 좁아진다.
> 숨을 고르면(감속) 온도가 내려가고, 안 보이던 반딧불이가 보인다.

## 실행

단일 HTML — `index.html`을 브라우저로 열면 끝. (로컬 서버: `python3 -m http.server 8941 --directory .`)

## 조작

| 입력 | 동작 |
|------|------|
| ← → / A D / 화면 좌·우 터치 | 조향 |
| ↑ / W / 부스트 버튼(꾹) | 가속 — 빠르지만 마음 온도가 빨리 오름 |
| SPACE / 숨 고르기 버튼(꾹) | 호흡 — 감속 + 마음 온도 하강 |
| 선택 카드 탭 | 해당 차선으로 자동 조향 |

## 과제 유형 (5종)

1. **회피** — 검붉은 자동사고 덩어리("어차피 난 안 돼" 등) 피하기. 충돌 시 온도 +16°
2. **호흡 존** — 8초 호흡 리듬(들숨/날숨)에 버튼 싱크 맞추기. 싱크가 맞을 때만 보너스 반딧불이 수집 가능
3. **갈림길** — 상황 제시 후 건강한 재해석 vs 인지왜곡, 좌/우 길 선택
4. **인지왜곡 관문** — 생각 문장을 보고 이름표(파국화·독심술·흑백논리·과잉일반화·개인화) 맞는 문으로 통과
5. **감정 이름 붙이기** — 빛의 곤충이 제시하는 상황에 가장 가까운 감정 어휘 선택

+ **과열 정지**: 마음 온도 100° 도달 시 강제 정차 → 호흡 3회로 재출발 (실패가 아니라 '숨 고르기'로 프레이밍)

## 기술

- 단일 HTML + Canvas 2D 유사3D(OutRun 방식 세그먼트 렌더링), 외부 라이브러리 없음
- WebAudio 신스 SFX + 앰비언트 패드, 진동 피드백(모바일)
- 콘텐츠 풀(갈림길 5·인지왜곡 5·감정 5)에서 매 판 셔플 출제
- **실사 에셋** (`assets/`, Higgsfield 생성 — 없으면 코드 그래픽으로 자동 폴백):
  - `intro.mp4` — Seedance 2.0 인트로 시네마틱(8초, 1080p, 오디오 포함). 첫 출발 시 1회 재생(건너뛰기 가능)
  - `sky.jpg` — 밤하늘+오로라+수평선 버섯 실루엣 배경
  - `mushroom.png` — 야광 버섯(핑크 원본, 캔버스 hue-rotate로 민트/시안/퍼플 변형 자동 생성)
  - `obstacle.png` 가시 생각덩어리 / `insect.png` 감정 곤충 / `handlebar.png` 1인칭 핸들바(하단 62% 크롭, 게이지 위치 `BAR` 상수)
